FirstNote Documentation

 

 

How to use FirstNote ?

FirstNote is a simple note software made for programmers (especially beginner programmers) and tutorial creators. This app is made to be easy to use.

Lets look at the main window:

The main window contains a few things. First, the title, which is FirstNote, obviously. It also contains the application version, just in case. The big square in the middle is the box. It contains all of your notes. As you can see, there is actually five notes here, and you can open them, edit them etc. but well get to that later.

Settings Button

The Settings button opens the Settings window. In this window, you can modify the note path, and do many more things.

List Button

The List button shows you where your notes are by opening the File Explorer for you.

As you can see here, it opened the File Explorer and I can see where my notes are! This is very useful because this software is based on sharing notes, so if someone shares you theirs note file, then you can simply put them here! You can also open the notes with Visual Studio Code or any other code editor or text editor to view them, but it wont be as good-looking as in FirstNote..

Reload Button

The Reload button allows you to reload the list. FirstNote is made for being light and not taking much performance. Thats why the Note List doesnt actually refreshes by itself. When you add a new file to your note folder, you will not see it directly on your list, you will have to reload it first.

However, you can make this automatic by going into:

-         Settings > Automatically refresh note list > Yes

New Note Button

You may have guessed, but this + button is made for creating new notes! Its very simple. When you click on it, the Note Window will appear:

This may be a little bit intimidating, but dont worry, its very simple to use.

The title section is where you put your note title.

The big box under it is the Rich Text Box. This box is just here to put a note in it.

 

As you can see at the top-left corner, there is a little menu:

The File one allows you to:

-         Save your note

-         Quit the note editor

 

The FirstNote one is a little more complex:

As you can see, the menu is pretty big, and there will surely have more with the updates soon.

Lets start with the basics.

The must-to-know ones are:

-         Place a Script Box

-         Place a Picture Box

-         Script Syntax (if you are scripting in your note)

 

First, lets focus on the script ones.

If you add a script box, your note will have a little script box under the Rich Text Box. It will allow you to put code snippets, examples and more. You can even configure the script language (syntax) as well see later.

You can see that the window now has the Script Box.

You can, in this Script Box, script in many different languages by changing the script Syntax:

There is CSharp, VB, HTML, XML, SQL, PHP, JS and Lua, but therell be more with the updates.

With Copy Rich Text (F2) and Copy Script (F1), you can actually copy the rich text and the script simply by pressing the buttons or shortcuts. And yeah, the key for copying script is F1, not F3. However, with F3 in the script text box, you can search for a function, or variable and more.

With Toggle Top Most,you can basically set the window in the first plan so you can stop worrying about it unfocusing when you get back on your code editor or any activity. It will stay here.

With Place a Picture Box, you can do the same as the script box, but with a picture box. This allows you to put a image instead of a script. Better for the UI examples if you are working in Unity, for example.

With Toggle Script Box and Picture Box, you can toggle the Script Box and the Picture Box, its as simple as that.

Now, lets see how to use the PictureBox. Dont worry, its easy.

Here is the window with the picture box.

The default picture is just the FirstNote base image. You can change it by pressing the

Button.

The Select an image button will open a dialog that allows you to choose your image.

When you select an image, double click it or click Open to put it in the PictureBox.

With the image selected and opened with the PictureBox, you have one more option to deal with. And it is the Size Mode.

Usually, the image will fit well, but sometimes, you will prefer to zoom it or stretch it. Its useful when you want to edit your image, or zoom on it, for example.

 

When you double click on the picture, it will open it in the Pictures software for you.
